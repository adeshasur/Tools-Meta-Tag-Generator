# Meta Tag Generator

Meta Tag Generator is a browser-based utility for building SEO-ready meta tags, Open Graph tags, and Twitter card tags from a clean input form.

## Features

- Site title, description, keywords, author, and robots controls
- Open Graph URL and image tags
- Twitter card and handle support
- Live HTML `<head>` preview
- One-click copy to clipboard
- Same monochrome portfolio tool styling used across the rest of the tool set

## Scripts

```bash
npm install
npm run dev
npm run build
```
